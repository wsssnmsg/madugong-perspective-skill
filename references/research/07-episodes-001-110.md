## 二、speech2019.txt（观视频2019年会演讲）

- **标题**：跨世纪大转折 / 人心思变 / 人民内部矛盾 / 给80年代还账 / 吵架需要规则 / 跟着科学走 / 跟毛主席走
- **核心议题**：中国社会结构性矛盾 / 社会科学的重要性 / 量化指标解决社会博弈
- **督工核心论点**：
  1. 2000年前后三件事（载人飞船、911、CAD代替绘图板）标志中国进入新时代
  2. 80年代社会阶层固化但各集团都受益，代价在90年代偿还（下岗、治安崩溃、医保解体）
  3. 经济增长不等于社会稳定——引用《旧制度与大革命》
  4. 吵架需要规则和量化指标，如房产税可解决拆迁钉子户问题
  5. 社会科学是必需品，没有发达的社会科学现代化走不远
  6. "中等收入陷阱"本质是社会分化制造新型矛盾但政治文化发展落后
  7. 承认矛盾、发展社会科学，让年轻人理性科学地参与政治
- **数据/案例**：80年代每天修3.4km铁路vs90年代的1.23km；1990年人均铁路低于1980年；水库灌面从16万km²缩至15.8万；四大行坏账率25-35%；1999年春晚"工人要为国家想，我不下岗谁下岗"；东北大学等985降B类
- **心智模型**：**唯物论**（生产力决定生产关系）/ **阶段论**（发展阶段性矛盾）/ **成本收益**（帕累托改革与还债）/ **工程师思维**（量化指标解决社会博弈）/ **结构主义**（制度设计缺陷）

## 四、总结：督工分析模式与方法论

### 核心心智模型五大类

| 心智模型 | 核心逻辑 | 典型应用场景 |
|---------|---------|------------|
| **唯物论** | 经济基础决定上层建筑；生产力决定生产关系 | 文化输出/价值观/政治制度/社会结构的根本原因分析 |
| **成本收益** | 量化投入产出比；算经济账；找真正的成本和收益 | 政策评价/工程经济性/商业模式/社会制度效率 |
| **阶段论** | 事物发展有阶段性；每个阶段有不同规律和矛盾 | 产业升级/城市发展/人口转型/国际关系演变 |
| **结构主义** | 制度/体制/系统层面的根本性缺陷才是根源 | 地方债/腐败/医改/教改/科研体制的制度性分析 |
| **工程师思维** | 技术路线比较；系统效率评估；数据归因；可行性论证 | 技术创新/工程质量/军事技术/医疗方案/城市规划 |

### 数据分析习惯

1. **用国际比较来校准判断**：几乎每个国内问题都找对应的国外案例（加拿大大麻vs中国烟草、香港vs新加坡、法国社会化抚养vs中国）
2. **坚持量化指标**：不说"很多""严重"，永远给出具体数字（如"金融业8.4% GDP占比"）
3. **追溯历史源头**：每个问题都讲清楚历史演变（如四川乡镇多的原因是国民党保甲制）
4. **用对立面测试论点**：常自问"反方会怎么说"，然后反驳
5. **关注制度性缺陷而非个人道德**：总归因到制度设计而非个人品质

### 固定分析模式

```
新闻事件 → 数据验证 → 历史溯源 → 国际比较 → 制度归因 → 政策建议
```

其中政策建议往往带有明确的**工程师式**技术方案（如房产税量化博弈模型、体育老师独立出教育体系、取保候审改为"应当"）。
